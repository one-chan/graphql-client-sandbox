## What is this?

The sandbox of GraphQL client for React using Github API and appllo.

## How to USE

### Initial settings

1. clone this.
1. install modules with `yarn`
1. set Github API token to environment variable as `GITHUB_GRAPHQL_API_TOKEN`.

### Generate code

1. run `yarn codegen`
